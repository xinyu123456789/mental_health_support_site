## ⚠️ 關鍵步驟：設定 AI 陪伴聊天的 API Key

本系統的「AI 聊天機器人」核心功能依賴 Groq API 運作。為了讓系統順利啟動並正確回覆訊息，**請使用者務必依照以下步驟，親自插入您的 API Key**：

### 📍 步驟 1：取得您的 Groq API Key
1. 前往 [Groq 官方網站](https://groq.com)
2. 註冊或登入您的帳號。
3. 點擊導覽列的 API Keys
4. 點擊畫面上的 **「Create API Key」** 按鈕。
5. 複製生成的那串金鑰（請妥善保存，它只會顯示一次）。

### 📍 步驟 2：找到專案中要「插進去」的地方
1. 請打開您剛剛下載下來的專案資料夾，尋找一個名為 `ai_chatbot` 的資料夾。

2. 尋找 `ai_chatbot` 資料夾裡面，一個名為 `service` 的資料夾。

3. 尋找 `service` 資料夾裡面，一個名為 `llm_client.py` 的檔案。

4. 在 `llm_client.py` 這個檔案裡面的第 10 行，找到 `api_key` 這個變數，把後面單引號內的 `YOUR GROQ API KEY` 改成你自己的 Gorq api key。


# 心理健康支援平台 🌿

這是一個專為提供心理支持與自我成長所設計的 Web 應用程式。本專案透過 AI 陪伴，協助使用者在低潮時獲得支持，並建立正向的心理韌性。

## 🌟 核心功能

* **AI 陪伴聊天：** 串接大型語言模型，具備共情能力的對話機器人。系統會計算情緒分數，並能在偵測到高風險語意時觸發 SOS 危機預警，切換至緊急應對風格。
* **每日誇誇筆記：** 鼓勵使用者每天存入一句讚美自己的話，建立正向思考習慣。
